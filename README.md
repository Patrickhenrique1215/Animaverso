# **Animaverso**

Plataforma web de streaming de animes desenvolvida como Single Page Application (SPA), oferecendo uma experiência imersiva e responsiva para os amantes de animação ocidental. Este projeto demonstra habilidades sólidas em desenvolvimento front-end utilizando React.

Demonstrar proficiência em:

- **Sass** 
- **Design responsivo**
